## 2일차 (2023-02-21)
  1. 이벤트 동기화
    - 이벤트 방식  : 이벤트 개체를 사용하여 대기 중인 쓰레드에 이벤트 발생을 알리는 방법으로 처리.
    - 순서 : 이벤트 생성 -> 쓰레드에 이벤트 전달 - > 메인 쓰레드 에서 이벤트 감시 -> setevent - > 메인쓰레드 에서 이벤트 감시 함수 반환
             -> 메인쓰레드 진행 

## 3일차 (2023-02-22)

    - UserMode 에서는 스트림,  KernelMode TCP - 세그먼트,  IP - 패킷 단위,
